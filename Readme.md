m# NYC GoGreen Tours – Official Website

The NYC GoGreen Tours website is the official digital platform for presenting and booking eco‑friendly tourism experiences across New York City. The site provides detailed tour information, company background, media galleries, and a streamlined booking interface for both VR Self‑Guided and Tour Guide experiences.

## Overview
NYC GoGreen Tours delivers sustainable, high‑quality tourism services designed for individuals, families, and groups seeking authentic New York City experiences. This website serves as the primary customer‑facing asset for marketing, information delivery, and reservation intake.

## Key Features
- Multi‑page responsive website optimized for desktop and mobile  
- Professional navigation with active page highlighting  
- VR Self‑Guided and Tour Guide booking options  
- Dedicated sections for tours, gallery, famous guests, and company history  
- SEO‑optimized metadata and structured schema for search visibility  
- Clean, business‑grade UI/UX with consistent branding  

## File Structure
- `index.html` — Home page  
- `about.html` — Company background and mission  
- `tours.html` — Complete tour catalog  
- `famous-guests.html` — Notable clients and VIPs  
- `gallery.html` — Media and photography showcase  
- `booking.html` — Reservation form and tour selection  
- `styles.css` — Global styling and responsive layout  
- `media/` — Images, icons, and brand assets  

## Media Content
The `media` directory contains all visual and branding assets used throughout the website, including:

- **Logo files**  
- **Tour photos**  
- **Gallery images**  
- **VR icons and Guide icons**  
- **Founder and staff photos**  
- **Marketing and promotional images**

All media files are organized into subfolders for clarity and scalability.

## Technology
