# NYC GoGreen Tours Website

A complete, multi‑page website for NYC GoGreen Tours, featuring eco‑friendly tour options including Walking, Bike, Pedicab, Horse & Carriage, VR Self‑Guided, and Tour Guide experiences.

## 📁 Project Structure
- `index.html` – Home page  
- `about.html` – About Us  
- `tours.html` – Tour Options  
- `famous-guests.html` – Celebrity & VIP Guests  
- `gallery.html` – Photo Gallery  
- `booking.html` – Booking Form with VR & Guide options  
- `styles.css` – Full site styling + mobile responsive layout  
- `media/` – Images, icons, and assets  

## 🚀 Features
- Fully responsive layout  
- Clean navigation with active page highlighting  
- VR vs Tour Guide comparison section  
- Highlight boxes for VR and Tour Guide  
- SEO‑optimized meta tags and schema  
- Organized media folder structure
